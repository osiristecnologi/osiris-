AULA 2 DE 10

HTML Básico - Estruturando Conteúdo

Aprenda as principais tags HTML para criar páginas completas

Títulos e Parágrafos

O HTML possui 6 níveis de títulos e tags para textos:

titulos.html 
<!DOCTYPE html>
<html>
<head>
    <title>Títulos e Parágrafos</title>
</head>
<body>
    <h1>Título Principal (H1)</h1>
    <h2>Subtítulo (H2)</h2>
    <h3>Seção (H3)</h3>
    <h4>Subseção (H4)</h4>
    <h5>Tópico (H5)</h5>
    <h6>Subtópico (H6)</h6>
    
    <p>Este é um parágrafo de texto.</p>
    <p>Este é outro parágrafo.</p>
    
    <p>Texto com <br> quebra de linha</p>
    
    <hr> <!-- Linha horizontal -->
</body>
</html>

Importante:

  • Use apenas UM <h1> por página (título principal)
  • Siga a hierarquia: H1 → H2 → H3 (não pule níveis)
  • <p> cria blocos de texto separados
  • <br> quebra linha SEM criar novo parágrafo

Listas

Existem 3 tipos de listas no HTML:

listas.html 
<!-- Lista Não Ordenada (bolinhas) -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>

<!-- Lista Ordenada (números) -->
<ol>
    <li>Primeiro passo</li>
    <li>Segundo passo</li>
    <li>Terceiro passo</li>
</ol>

<!-- Lista de Definição -->
<dl>
    <dt>HTML</dt>
    <dd>Linguagem de marcação</dd>
    
    <dt>CSS</dt>
    <dd>Linguagem de estilo</dd>
</dl>

Links e Imagens

Conecte páginas e adicione imagens:

links-imagens.html 
<!-- Links -->
<a href="https://google.com">Ir para Google</a>

<a href="pagina2.html">Ir para outra página</a>

<a href="#topo">Ir para topo da página</a>

<a href="arquivo.pdf" download>Baixar PDF</a>

<!-- Link abre em nova aba -->
<a href="https://google.com" target="_blank">
    Google (nova aba)
</a>

<!-- Imagens -->
<img src="foto.jpg" alt="Descrição da imagem">

<img src="logo.png" 
     alt="Logo OSIRIS" 
     width="200" 
     height="100">

<!-- Imagem da internet -->
<img src="https://exemplo.com/imagem.jpg" 
     alt="Imagem externa">

Atributos Importantes:

  • href = destino do link
  • target="_blank" = abre em nova aba
  • src = caminho da imagem
  • alt = texto alternativo (acessibilidade)
  • width/height = largura/altura

Tabelas

Crie tabelas para organizar dados:

tabela.html 
<table border="1">
    <thead>
        <tr>
            <th>Nome</th>
            <th>Idade</th>
            <th>Cidade</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>João</td>
            <td>25</td>
            <td>São Paulo</td>
        </tr>
        <tr>
            <td>Maria</td>
            <td>30</td>
            <td>Rio de Janeiro</td>
        </tr>
    </tbody>
</table>

Estrutura da Tabela:

  • <table> - container da tabela
  • <thead> - cabeçalho
  • <tbody> - corpo (dados)
  • <tr> - linha (table row)
  • <th> - célula de título (header)
  • <td> - célula de dado (data)

Formulários

Colete dados dos usuários:

formulario.html 
<form action="/enviar" method="POST">
    
    <label for="nome">Nome:</label>
    <input type="text" id="nome" name="nome" required>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    
    <label for="senha">Senha:</label>
    <input type="password" id="senha" name="senha">
    
    <label for="idade">Idade:</label>
    <input type="number" id="idade" name="idade">
    
    <label>Gênero:</label>
    <input type="radio" name="genero" value="m"> Masculino
    <input type="radio" name="genero" value="f"> Feminino
    
    <label>Interesses:</label>
    <input type="checkbox" name="interesse" value="html"> HTML
    <input type="checkbox" name="interesse" value="css"> CSS
    <input type="checkbox" name="interesse" value="js"> JavaScript
    
    <label for="pais">País:</label>
    <select id="pais" name="pais">
        <option value="">Selecione...</option>
        <option value="br">Brasil</option>
        <option value="pt">Portugal</option>
        <option value="us">EUA</option>
    </select>
    
    <label for="mensagem">Mensagem:</label>
    <textarea id="mensagem" name="mensagem" rows="4"></textarea>
    
    <button type="submit">Enviar</button>
    <button type="reset">Limpar</button>
    
</form>

Tipos de Input:

  • text - texto simples
  • email - valida email
  • password - esconde texto
  • number - apenas números
  • radio - escolha única
  • checkbox - múltipla escolha
  • date - seletor de data
  • file - upload de arquivo

Divisões e Containers

Organize o conteúdo em blocos:

divs.html 
<!-- Div genérica -->
<div class="container">
    <div class="cabecalho">
        <h1>Meu Site</h1>
    </div>
    
    <div class="conteudo">
        <p>Conteúdo principal</p>
    </div>
    
    <div class="rodape">
        <p>Rodapé</p>
    </div>
</div>

<!-- Span (inline) -->
<p>Texto com <span style="color: red">parte colorida</span></p>

Diferença:

  • <div> = bloco (ocupa largura total)
  • <span> = inline (ocupa só o necessário)

Exercício Prático

Desafio da Aula 2:

Crie uma página de Currículo com:

  • ✓ Seu nome em <h1>
  • ✓ Dados pessoais (email, telefone) em lista
  • ✓ Experiência profissional em <ol>
  • ✓ Habilidades em <ul>
  • ✓ Tabela de formação acadêmica
  • ✓ Link para LinkedIn
  • ✓ Foto sua (use <img>)

Exemplo de estrutura:

<h1>Seu Nome</h1>
<img src="foto.jpg" alt="Minha foto">

<h2>Contato</h2>
<ul>
  <li>Email: seu@email.com</li>
  <li>Tel: (11) 99999-9999</li>
</ul>

<h2>Experiência</h2>
<ol>
  <li>Empresa A - 2020-2022</li>
  <li>Empresa B - 2022-atual</li>
</ol>

Resumo da Aula

  • H1-H6 para títulos hierárquicos
  • <p> para parágrafos
  • <ul>, <ol>, <dl> para listas
  • <a href=""> para links
  • <img src="" alt=""> para imagens
  • <table> para tabelas
  • <form> com <input> para formulários
  • <div> e <span> para organizar
