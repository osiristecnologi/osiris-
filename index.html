Introdução à Programação Web

Entenda como a web funciona e prepare seu ambiente

O que é Programação?

Programar é dar instruções para um computador executar tarefas específicas. É como escrever uma receita de bolo, mas para máquinas!

Analogia Simples:

  • Você: "Pegue o copo d'água"
  • Computador: Precisa de instruções detalhadas:
    1. Localizar o copo
    2. Mover a mão até o copo
    3. Agarrar o copo
    4. Levantar o copo

Como Funciona a Web?

A web é baseada em três tecnologias principais:

HTML

Estrutura - É o esqueleto da página. Define o conteúdo (títulos, parágrafos, imagens).

<h1>Olá Mundo</h1>

CSS

Estilo - É a "roupa" da página. Define cores, fontes, layout.

h1 { color: blue; }

JavaScript

Comportamento - É o "cérebro". Torna a página interativa.

alert('Olá!');

Configurando seu Ambiente

Você precisa de apenas 2 coisas para começar:

1

Editor de Código

Recomendamos o VS Code (Visual Studio Code):

  • Baixe em: code.visualstudio.com
  • É gratuito e leve
  • Funciona em Windows, Mac e Linux
2

Navegador Web

Use o Google Chrome ou Firefox:

  • Abra o arquivo HTML neles
  • Use o Inspecionar (F12) para ver o código

Seu Primeiro Código

Vamos criar sua primeira página HTML!

Passo a Passo:

  1. Abra o VS Code
  2. Crie um novo arquivo (Ctrl+N)
  3. Digite o código abaixo:
meu-primeiro-site.html 
<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Primeira Página</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Esta é minha primeira página web!</p>
    <p>Estou aprendendo programação no OSIRIS! 👁️</p>
</body>
</html>
  1. Salve o arquivo como meu-primeiro-site.html
  2. Abra o arquivo no navegador (duplo clique)
  3. Pronto! Você verá sua página!

Resumo da Aula

  • Programação = Dar instruções ao computador
  • HTML = Estrutura | CSS = Estilo | JS = Comportamento
  • VS Code é o editor recomendado
  • HTML usa tags entre < >
  • Todo arquivo HTML precisa de <html>, <head> e <body>
